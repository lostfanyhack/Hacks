# Install ponysay to your system:

## MacOS:
### Install Homebrew (https://brew.sh)
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
### Install ponysay
```
brew install ponysay
```
## Linux:
```
sudo apt-get install ponysay
```
# Run it:
```
ponysay Hello World
```
