# Install nsnake to your system:

## MacOS:
### Install Homebrew (https://brew.sh)
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
### Install nsnake
```
brew install nsnake
```
## Linux:
```
sudo apt-get install nsnake
```
# Run it:
```
nsnake
```
